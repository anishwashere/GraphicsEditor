# GraphicsEditor

A menu-driven 2D Graphics Editor written in C using a 2D character array as the drawing canvas.

## Features

- Draw Rectangle
- Draw Line
- Draw Circle
- Draw Triangle
- Add Shapes
- Delete Shapes
- Modify Shapes
- List Shapes
- Display Canvas

## Technologies Used

- C Programming
- Arrays
- Structures
- Functions
- Git & GitHub
